# 1.8.0

_In development. Entries below are populated as tasks complete; final list compiled at release._

## Features
- Settings and the downloaded `data/` tree (templates, dependencies, Qt modules, stylesheet, scratch) now live in the per-application AppData folder (`%AppData%\Roaming\KROIA\CmakeLibCreator\` on Windows). Wiping the build folder no longer destroys settings.
- New **Project Paths** editor inside the existing Settings dialog: add, remove, and browse for the project paths previously only editable by hand-editing `settings.json`.
- New **Default library path** setting (Settings dialog). Browse buttons inside the editor and the "Open existing library" / "Save as new" file dialogs all default to this path when no project is loaded yet. Default value: `<Documents>\Visual Studio 2022\Projects`.
